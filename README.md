# لوحة إدارة قنوات Kattaa TV على GitHub Pages

لوحة ويب لإدارة ملف `channels.json` الذي يقرأه تطبيق التلفاز عند ضغط زر **«تحديث»**.

## الميزات

- عرض جميع القنوات مع البحث والتصفية والترتيب.
- **معاينة البث داخل الصفحة** (زر ▶ معاينة) بمشغّل HLS للتأكد من أن القناة تعمل قبل نشرها.
- **فحص كل القنوات** بضغطة واحدة: تظهر حالة كل قناة (يعمل ✓ / لا يعمل ✗ / غير مؤكد).
- إضافة وتعديل وحذف القنوات، مع معاينة الرابط من داخل نافذة التعديل.
- **حفظ مباشر إلى GitHub** من الصفحة نفسها (برمز وصول يُحفظ في متصفحك فقط)،
  أو تنزيل الملف ورفعه يدويًا.
- حفظ تلقائي للمسودة في المتصفح كي لا تضيع تعديلاتك.

## الرفع إلى GitHub وتفعيل الصفحة (مرة واحدة)

1. أنشئ مستودعًا جديدًا على GitHub (عامًّا)، مثلًا باسم `kattaa-channels`.
2. ارفع `index.html` و`channels.json` إلى الفرع `main`.
3. افتح **Settings ← Pages** واختر:
   - Source: **Deploy from a branch**
   - Branch: **main** — Folder: **/ (root)**
4. بعد دقيقة تصبح اللوحة متاحة على:
   `https://USERNAME.github.io/kattaa-channels/`
   وملف القنوات على:
   `https://USERNAME.github.io/kattaa-channels/channels.json`

## ربط تطبيق التلفاز (مرة واحدة)

في التطبيق: **OK ← الإعدادات ← مصدر قائمة القنوات** وأدخل رابط
`channels.json` أعلاه ثم «حفظ القائمة وتحميلها».

بعد ذلك، كلما عدّلت القنوات هنا وحفظت، اضغط زر **«تحديث»** في التطبيق
(موجود في قائمة القنوات وفي الإعدادات) فتصل القائمة الجديدة فورًا.
التطبيق يحفظ آخر قائمة منزّلة على الجهاز، ويحاول تحديثها تلقائيًا بصمت عند كل تشغيل.

> بديل: يمكن تثبيت الرابط داخل التطبيق نفسه بتعديل الثابت `DEFAULT_REMOTE_URL`
> في `app/src/main/java/com/arabtv/data/source/ChannelRepository.kt` ثم إعادة بناء APK —
> عندها لا حاجة لإدخال الرابط في الإعدادات.

## الحفظ المباشر إلى GitHub (اختياري لكنه الأسهل)

1. من GitHub: **Settings ← Developer settings ← Fine-grained personal access tokens ← Generate new token**.
2. اقصر الرمز على مستودع القنوات فقط، وامنحه صلاحية **Contents: Read and write** لا غير.
3. في اللوحة: **⚙ إعدادات GitHub** وأدخل اسم المستخدم والمستودع والرمز.
4. صار زر **☁ حفظ إلى GitHub** يكتب `channels.json` في المستودع مباشرة (commit).

> الرمز يُحفظ في متصفح جهازك فقط (localStorage) ولا يُرفع مع الصفحة.
> لا تدخل الرمز على جهاز عام، ويمكنك مسحه بزر «مسح الرمز».

## ملاحظات الفحص والمعاينة

- المعاينة تشغّل البث فعليًا في المتصفح — أفضل تأكيد على أن القناة تعمل.
- بعض الخوادم تحجب التشغيل من المتصفح (CORS) رغم أنها تعمل داخل التطبيق؛
  تظهر هذه الحالة كـ«غير مؤكد».
- روابط `http://` (غير المشفرة) لا يستطيع المتصفح اختبارها من صفحة `https`
  (محتوى مختلط)، لكنها قد تعمل في التطبيق على التلفاز.

## تشغيل محلي للاختبار

لا تفتح `index.html` بالنقر المزدوج، لأن المتصفح قد يمنع `fetch` للملف المحلي.
شغّل خادمًا بسيطًا داخل المجلد:

```bash
python -m http.server 8000
```

ثم افتح: `http://localhost:8000`
